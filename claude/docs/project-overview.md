# Visão Geral do Projeto

## Mandure Serviços Landing Page

Landing page institucional para empreiteira. Foco em:
- **Conversão**: Captação via WhatsApp
- **Showcase**: Portfolio com animações premium
- **Credibilidade**: Estatísticas, depoimentos, certificações

## Stack

- Next.js 15 (App Router) + TypeScript strict
- Tailwind CSS + Design Tokens Bronze/Slate
- Framer Motion (efeitos arquitetônicos)
- Playwright E2E
- Docker + GitHub Actions

## Seções (9)

1. Hero — Background imersivo, CTAs WhatsApp
2. Stats — Contadores animados (4 métricas)
3. Services — 4 serviços com cards
4. Portfolio — Galeria filtrável por categoria
5. Process — Timeline de 4 etapas
6. Testimonials — 3 depoimentos em grid
7. Certifications — Certificações CREA, ISO, PBQP-H e CBIC em cards informativos
8. CTAFinal — Bronze section com CTAs
9. WhatsAppButton — Botão fixo com pulsação

## Métricas de Sucesso

- Lighthouse Performance > 90
- LCP < 2.5s
- WCAG AA acessibilidade
- TypeScript 0 erros
