# Visao Geral do Projeto

## Mandure Servicos Landing Page

Landing page institucional para empreiteira com foco em:
- Conversao via WhatsApp
- Showcase de portfolio com animacoes
- Credibilidade com dados, depoimentos e certificacoes

## Stack
- Next.js (App Router) + TypeScript strict
- Tailwind CSS + design tokens Bronze/Slate
- Framer Motion
- Playwright E2E
- Docker + GitHub Actions

## Secoes principais
1. Hero
2. Stats
3. Services
4. Portfolio
5. Process
6. Testimonials
7. Certifications
8. CTAFinal
9. WhatsAppButton

## Metricas alvo
- Lighthouse Performance > 90
- LCP < 2.5s
- WCAG AA
- TypeScript com 0 erros
